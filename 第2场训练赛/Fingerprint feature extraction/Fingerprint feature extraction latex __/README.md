# cumcmthesis


## Description

CUMCU LaTeX Template

2016年[*全国大学生数学建模竞赛*](http://www.mcm.edu.cn/) LaTeX模板

## Note

- 环境：Texlive2016
- UTF-8 编码
- 使用 XeLaTeX编译
- 提供了`clean.bat`和`thesis.bat`两个脚本分别用于临时文件和输出pdf
- 基于2015年的格式要求
